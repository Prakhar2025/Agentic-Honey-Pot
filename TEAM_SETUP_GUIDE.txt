   - Click "GET /v1/analytics/dashboard"

â“ TROUBLESHOOTING
--------------------------------------------------------------------------------

Q: "ModuleNotFoundError: No module named 'X'"
A: Run: pip install -r requirements.txt

Q: "Address already in use"
A: Another process using port 8000. Kill it or use: 
   uvicorn app.main:app --reload --port 8001

Q: "GROQ_API_KEY not found"
A: Make sure .env file exists with GROQ_API_KEY=gsk_...

Q: API responds slowly (>3 seconds)
A: Normal! LLM calls take 1-3 seconds. It's acceptable.

Q: "Database locked"
A: Delete data/scamshield.db and restart server
