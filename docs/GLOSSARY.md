## Technical Terms

| Term | Definition |
|------|------------|
| **Agentic AI** | AI system that autonomously takes actions to achieve goals |
| **API** | Application Programming Interface - how software components communicate |
| **Async** | Asynchronous programming - non-blocking operations |
| **Bearer Token** | Authentication credential passed in HTTP header |
| **CORS** | Cross-Origin Resource Sharing - browser security feature |
| **Endpoint** | Specific URL path that handles API requests |
| **FastAPI** | Modern Python web framework for building APIs |
| **Groq** | LLM inference provider with fast response times |
| **LLM** | Large Language Model - AI trained on text data |
| **Middleware** | Code that runs between request and response |
| **ORM** | Object-Relational Mapping - database abstraction |
| **Pydantic** | Python library for data validation |
| **REST** | REpresentational State Transfer - API design style |
| **SQLAlchemy** | Python SQL toolkit and ORM |
| **UUID** | Universally Unique Identifier |

## Domain Terms (Scam Types)

| Term | Definition |
|------|------------|
| **KYC Fraud** | Scam claiming Know Your Customer verification is needed |
| **Lottery Scam** | Fake prize/lottery winning notification |
| **Tech Support Scam** | Fake technical support claiming device issues |
| **Investment Fraud** | Fake investment opportunity promising returns |
| **Job Scam** | Fake job offer requiring upfront payment |
| **Loan Scam** | Fake loan approval requiring processing fee |
| **OTP Fraud** | Attempt to steal One-Time Password |
| **Phishing** | Fraudulent attempt to obtain sensitive information |
| **Vishing** | Voice phishing - phone-based scam |
| **Smishing** | SMS phishing - text message scam |

## Session States

| State | Definition |
|-------|------------|
| **INITIAL** | Session just created |
| **ONGOING** | Active conversation in progress |
| **COMPLETED** | Successfully concluded |
| **TERMINATED** | Scammer stopped responding |
| **MAX_TURNS_REACHED** | Hit conversation limit |
